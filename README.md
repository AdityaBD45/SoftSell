# SoftSell - Sell Your Software Licenses

**SoftSell** is a one-page, responsive marketing website designed to help users sell unused software licenses. The platform ensures a fast, legal, and hassle-free process. Whether you're an individual or a business, SoftSell offers an easy solution to monetize software licenses that are no longer in use.

## Features

- **Responsive Design:** A fully responsive layout that works seamlessly on both desktop and mobile devices.
- **Dark Mode:** Built-in dark mode for a comfortable browsing experience, even in low-light environments.
- **Smooth Animations:** Enhanced user experience with animations powered by Framer Motion and AOS.
- **Interactive Components:** Engaging UI components using Tailwind CSS and Lucide-React icons.
- **Clear CTA Buttons:** Prominent call-to-action buttons that encourage users to take the next step.
**Live Chat:** Integrated **Tawk.to** live chat to help users get in touch with the support team instantly.


## Tech Stack

- **Frontend:**
  - React.js (for building dynamic user interfaces)
  - Tailwind CSS (for fast and efficient styling)
  - Framer Motion (for adding animations and transitions)
  - AOS (for scroll animations)
  - Lucide-React (for icons)

## Dark Mode

SoftSell supports dark mode for users who prefer a darker interface. This preference is saved using the browser's `localStorage`, ensuring that the user's choice persists across sessions.

To toggle dark mode, simply click the "dark mode" button located in the top navigation bar.
